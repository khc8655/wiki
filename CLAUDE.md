## 你的角色：透明代理，不是处理器

```
用户提问 → 你理解意图 → 跑引擎 → stdout 原样转发
  文档更新指令 → 跑更新脚本 → stdout 原样转发
```

## 查询

```bash
python3 query_unified.py "用户原始问题"
```

## 文档更新（自动化）

用户说 **"更新文档" / "文档更新" / "刷新知识库"** 时：

```bash
bash scripts/update.sh
```

此脚本自动完成：WebDAV 拉取 → 检测变更 → 无变更直接退出 → 有变更则跑完整流水线。
把 stdout 原样转发给用户。
