# Wiki 知识库 Schema

> 版本: v3.1 | 更新日期: 2026-05-04

## 知识库定位

小鱼易连业务知识库 —— 支持型号查询、方案检索、参数对比、版本追踪。

## 标签分类 (Tag Taxonomy)

### 意图标签 (intent_tags)

从以下分类中选择 1-3 个：

| 分类 | 含义 | 触发词示例 |
|------|------|-----------|
| 安全保障 | 加密、认证、合规 | 国密、SM2/3/4、TLS、等保 |
| 功能更新 | 新功能、版本迭代 | 迭代、新增、优化、发布 |
| 部署运维 | 安装、配置、运维 | 部署、安装、配置、升级 |
| 架构设计 | 系统架构、拓扑 | 架构、拓扑、高可用、容灾 |
| 集成对接 | API、SDK、对接 | API、SDK、接口、对接 |
| 场景方案 | 行业解决方案 | 公安、医疗、金融、政府 |
| 性能参数 | 规格、参数、指标 | 4K、1080P、延迟、并发 |
| 报价价格 | 价格、报价、成本 | 价格、报价、费用、▲ |
| 利旧兼容 | 兼容性、迁移 | 兼容、利旧、替代、升级 |
| 培训赋能 | 培训、文档、教程 | 培训、教程、指南、手册 |
| 接口硬件 | 接口、硬件规格 | HDMI、USB、XLR、网口 |

### 概念标签 (concept_tags)

技术概念，3-8 个。示例：
- 编解码: H.264, H.265, Opus, SVC
- 协议: SIP, H.323, WebRTC
- 安全: 国密SM4, TLS1.3, 端到端加密
- 架构: 私有化, 混合云, 多活热备

### 场景标签 (scenario_tags)

行业场景，可选。示例：
- 公安, 检察院, 法院, 司法
- 医疗, 教育, 金融
- 应急管理, 水利, 气象

### 型号标签 (models)

产品型号，从正文提取。匹配规则：
```
(AE\d{3}[A-Z]?|XE\d{3}[A-Z]?|GE\d{3}[A-Z]?|PE\d{4}|TP\d{3}(?:-[A-Z])?|MX\d{2}|AC\d{2}|NC\d{2}|NP\d{2}(?:V?\d+)?)
```

示例: AE800, XE800, GE600, PE8000, TP10, MX50

## 卡片结构

```json
{
  "id": "唯一标识",
  "doc_file": "来源文档",
  "title": "段落标题",
  "path": "文档路径层级",
  "body": "正文内容",
  "semantic": {
    "intent_tags": ["安全保障"],
    "concept_tags": ["国密SM4"],
    "scenario_tags": [],
    "models": ["AE700"],
    "keywords": ["端到端加密"]
  }
}
```

## 四源路由规则

| 查询特征 | 路由 | 检索方式 |
|----------|------|----------|
| 型号 + 价格/参数/招标/对比/接口 | excel | SQLite 列语义映射 |
| 概念/场景/架构/安全/部署 | knowledge | BM25 + Vector 混合 |
| 版本/迭代/新功能/发版 | update | BM25 粗粒度段落 |
| ppt/幻灯片 | ppt | 图片理解卡片 |

## 输出规范

查询输出 **不做总结**，直接返回原文：

```
标题: xxx
正文原文: xxx（完整段落，不截取）
出处: 文档名 | 路径层级
命中率: xx%
```

## 反馈闭环

- 每次查询记录到 `index_store/query_feedback.jsonl`
- 低质量（命中率<30%）触发优化建议
- 累积反馈自动调整 BM25/Vector 权重

## 更新策略

当原始文档变更时：
1. WebDAV 同步检测到 sha256 变化
2. 重新切段生成卡片
3. 增量标注新卡片
4. 重建向量索引
