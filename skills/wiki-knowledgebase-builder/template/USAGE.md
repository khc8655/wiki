# Template Usage

1. Copy `template/` into a new repository or workspace.
2. Replace placeholders like `{{PROJECT_NAME}}` and `{{PROJECT_DESC}}`.
3. Import source material into `raw/`.
4. Generate cards, metadata, and indexes.
5. Implement or copy query scripts.
6. Add routes, topics, and wiki synthesis pages.

This template is intentionally minimal. Extend it per corpus type.
