# Knowledge Tree V2

This folder contains a 4-layer prototype knowledge tree.

## Layers
1. L1: 大类（平台 / 终端）
2. L2: 主题（如安全、AI、稳定性、模块架构、产品组成、系统连接）
3. L3: 标题索引层
4. L4: 段落索引层（每段带 match_percent）

## Files
- `knowledge_tree.v2.json`: nested tree structure
- `title_index.v2.json`: flat title index
- `paragraph_index.v2.json`: flat paragraph index with percentages
- `stats.v2.json`: simple stats
