## 核心主线

### 1. 云平台功能增强
- 当前会议/历史会议支持呼叫服务器拓扑
- 自定义登录密码策略
- 账密 + 短信验证码登录
- 会议号自定义默认全量放开
- 告警中心增强
- 当前/历史会议导出未入会列表

### 2. 会控与布局管控增强
- 多画面/轮询能力增强
- 画中画与 48:9 特殊布局
- 多屏管理升级
- 点名/发言模式布局配置化

### 3. 终端与双模双活增强
- BYOM-NP30 v2
- Web 音频设置优化
- 配置导入/导出产品化
- 共享画面拉伸
- 轮询列表上限扩展
- 休眠/唤醒与定时重启

### 4. 运维与升级影响
- 服务缩减与合并
- 夜莺外链去除
- statis 服务增加数据库巡检
- AMS1000 proxyIp 接入策略调整
- manager / xyupgrade 支持版本回退
